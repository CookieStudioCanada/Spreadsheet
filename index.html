Help
1. Basic Operations
  • Click any cell to edit
  • Use arrow keys to navigate
  • Press Enter to confirm, Esc to cancel
2. Formulas (start with =)
Basic Math:
  • Addition: =A1+B1
  • Subtraction: =A1-B1
  • Multiplication: =A1*B1
  • Division: =A1/B1
  • Power: =POWER(A1,2) or =A1^2
  • Square Root: =SQRT(A1)
Statistical:
  • Sum: =SUM(A1:A5)
  • Average: =AVERAGE(A1:A5)
  • Count: =COUNT(A1:A5)
  • Maximum: =MAX(A1:A5)
  • Minimum: =MIN(A1:A5)
  • Median: =MEDIAN(A1:A5)
Logical:
  • IF: =IF(A1>10,"High","Low")
  • AND: =AND(A1>10,B1<20)
  • OR: =OR(A1>10,B1<20)
  • NOT: =NOT(A1>10)
Text:
  • Concatenate: =CONCATENATE(A1," ",B1)
  • Left: =LEFT(A1,3)
  • Right: =RIGHT(A1,3)
  • Length: =LEN(A1)
3. Features
  • Right-click for context menu
  • Drag column/row borders to resize
  • Double-click borders for auto-resize
4. Keyboard Shortcuts
  • Ctrl+C: Copy
  • Ctrl+V: Paste
  • Ctrl+Z: Undo
  • Ctrl+Y: Redo
